# 🎮 Memorix

Memorix is a browser-based memory game built using HTML, CSS, and JavaScript. Players must remember and repeat increasingly difficult sequences to progress through the levels.

## 🚀 Features

- Interactive 3×3 game board
- Random sequence generation
- Dynamic level progression
- Flash animations
- Game-over and restart system

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How to Play

1. Press any key to start the game.
2. Memorize the highlighted sequence.
3. Repeat the sequence by clicking the boxes.
4. Advance through the levels.
5. Try to achieve the highest score.
